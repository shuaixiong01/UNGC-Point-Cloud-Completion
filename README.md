UNGC: Unsupervised Neighbor-Guided Point Cloud Completion

[cite_start]This repository will contain the official PyTorch implementation for the paper "Unsupervised 3D Shape Completion via CLIP-Based Consistency"[cite: 1]. 

🚀 Status: Coming Soon!

Currently, the source code, pre-trained weights, and data preprocessing scripts are undergoing internal review and code clean-up. 

To ensure the best reproducibility and usability for the community, the complete codebase will be made publicly available immediately upon the formal acceptance of the paper.

📋 What will be included:
Complete PyTorch architecture of the UNGC framework.
Implementation of the Neighbor Search Module (NSM) and Density-Aware Gating Module (DAGM).
Training and evaluation scripts for EPN3D, ScanNet, and KITTI datasets.
Pre-trained model weights.
